# Árvores
Trabalho feito por Gian Victor Gonçalves Figueiredo e Felipe Lisboa Brasil.

Os arquivos estarão em Java, e os arquivos compilados estarão na pasta bin.

O arquivo `ArvoreRubroNegra.java` refere-se às Árvores Rubro Negras e o arquivo `Avlarquivo.java`refere-se às Árvores AVL.

# Compilar e Executar
Para compilar o código da árvore rubro negra, use `javac -d bin ArvoreRubroNegra.java` e para e executar use `java -cp bin ArvoreRubroNegra`

Para compilar o código da árvore rubro negra, use `javac -d bin Avlarquivo.java` e para e executar use `java -cp bin Avlarquivo`
